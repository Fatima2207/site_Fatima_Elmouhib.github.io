# 🎮 Game Zone - Projet Web

## 📌 Description du projet

Game Zone est un site web dédié aux jeux vidéo.  
Il permet de découvrir différents jeux populaires, lire des descriptions et consulter des avis de joueurs.

Ce projet a été réalisé dans le cadre d’un apprentissage du développement web (HTML, CSS et JavaScript).

---

## 🎯 Objectif du projet

- Créer un site web complet et responsive
- Appliquer les bases du développement web
- Utiliser JavaScript pour ajouter des interactions
- S’initier à la conception avec Figma

---

## 🛠️ Technologies utilisées

- HTML5 🧱
- CSS3 🎨
- JavaScript ⚙️
- Figma (design UI/UX) ✏️
